# Task 3: Deep-Dive Analysis & Interactive Dashboard

This project is part of my Data Analytics Internship at ApexPlanet Software Pvt. Ltd.

## Project Overview

The objective of this task was to perform a deep-dive business analysis and build an interactive dashboard to visualize key performance indicators (KPIs) and insights from the Superstore dataset.

## Tools Used

- Power BI
- Data Analytics Concepts
- Data Visualization

## Key Performance Indicators (KPIs)

- Total Sales
- Total Orders
- Average Shipping Days
- Average Order Value

## Dashboard Features

- Sales performance overview
- Monthly sales trend analysis
- Sales by category and region
- Top performing products
- Interactive filters (Segment, Year, Region, Date)

## Key Insights

- The West region generates the highest overall sales.
- Technology category contributes the largest share of revenue.
- Sales increase toward the end of the year, indicating seasonal demand.
- Consumer segment accounts for the majority of orders.
- Average shipping time remains consistent around 4 days.
